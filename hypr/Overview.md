# Hyprland Keyboard Shortcuts

Mod-Taste (`mainMod`) ist `SUPER` (Windows-Taste). Quelle: `config/binds.lua`.

## Fenster-Verwaltung

| Shortcut | Aktion |
|---|---|
| `SUPER + Escape` | Aktives Fenster killen (`hyprctl kill`) |
| `SUPER + Q` | Fenster schließen |
| `SUPER + ALT + Space` | Floating an/aus |
| `SUPER + D` | Fullscreen (Mode 1) |
| `SUPER + F` | Fullscreen |
| `SUPER + J` | Split-Richtung umschalten |

## Fokus wechseln

| Shortcut | Aktion |
|---|---|
| `SUPER + Links/Rechts/Hoch/Runter` | Fokus in Richtung bewegen |
| `ALT + Tab` | Zum nächsten Fenster wechseln |
| `SUPER + Tab` | Noctalia Window-Switcher öffnen |

## Fenster verschieben (Workspaces & Monitore)

| Shortcut | Aktion |
|---|---|
| `SUPER + SHIFT + Rechts/Links/Hoch/Runter` | Fenster in Richtung verschieben |
| `SUPER + SHIFT + 1/2/3` | Fenster zu Monitor 1/2/3 |
| `SUPER + SHIFT + Mausrad hoch` | Fenster zu Monitor +1 |
| `SUPER + SHIFT + Mausrad runter` | Fenster zu Monitor -1 |
| `SUPER + CONTROL + SHIFT + Rechts` | Fenster zu Workspace relativ +1 |
| `SUPER + CONTROL + SHIFT + Links` | Fenster zu Workspace relativ -1 |
| `SUPER + SHIFT + CONTROL + 1..N` | Fenster zu Workspace N auf aktuellem Monitor (N = `NUM_WPM`, aktuell 3) |

## Maus

| Shortcut | Aktion |
|---|---|
| `SUPER + Maustaste 8 (272)` | Fenster ziehen (drag) |
| `SUPER + Maustaste 9 (273)` | Fenster resizen |

## Launcher / Anwendungen

| Shortcut | Aktion |
|---|---|
| `SUPER + Return` | Terminal öffnen (`kitty`) |
| `SUPER + E` | Dateimanager öffnen (`dolphin`) |
| `SUPER + T` | Editor öffnen (`gnome-text-editor --new-window`) |
| `SUPER + C` | Taschenrechner öffnen (`gnome-calculator`) |
| `SUPER + W` | Browser öffnen (`firefox`) |
| `CONTROL + SHIFT + Escape` | Terminal mit `btop` öffnen |
| `SUPER + Z` | Noctalia Settings umschalten |
| `SUPER + X` | Noctalia Control-Center umschalten |
| `SUPER + Space` | Noctalia Launcher umschalten |
| `SUPER + .` (period) | Noctalia Launcher (Emoji-Modus `/emo`) |
| `SUPER + L` | Session sperren |
| `SUPER + ALT + C` | Noctalia Session-Panel umschalten |

## Hardware-Steuerung

| Shortcut | Aktion |
|---|---|
| `XF86AudioRaiseVolume` | Lautstärke hoch |
| `XF86AudioLowerVolume` | Lautstärke runter |
| `XF86AudioMute` | Stumm schalten |
| `XF86AudioMicMute` | Mikrofon stumm schalten |
| `XF86AudioPlay` / `XF86AudioPause` | Medienwiedergabe umschalten |
| `XF86AudioNext` | Nächster Titel |
| `XF86AudioPrev` | Vorheriger Titel |
| `XF86MonBrightnessUp` | Helligkeit hoch |
| `XF86MonBrightnessDown` | Helligkeit runter |

## Utilities

| Shortcut | Aktion |
|---|---|
| `SUPER + P` | Hyprpicker (Farbe von Bildschirm aufnehmen) |
| `Print` | Screenshot-Bereich auswählen |
| `SUPER + Print` | Vollbild-Screenshot |
| `SUPER + SHIFT + W` | Wallpaper-Panel umschalten |
| `SUPER + V` | Clipboard-Panel umschalten |
| `SUPER + A` | Benachrichtigungs-Center umschalten |

## Workspaces & Monitore

| Shortcut | Aktion |
|---|---|
| `SUPER + 1/2/3` | Fokus auf Monitor 1/2/3 |
| `SUPER + TAB + 1..N` | Fokus auf Workspace N (absolut, N = `NUM_WPM`, aktuell 3) |
| `SUPER + CONTROL + 1..N` | Fokus auf Workspace N relativ zum aktuellen Monitor |
| `SUPER + CONTROL + Rechts` | Fokus zu Monitor +1 |
| `SUPER + CONTROL + Links` | Fokus zu Monitor -1 |
| `SUPER + CONTROL + Runter` | Fokus zum nächsten leeren Monitor |
| `SUPER + Mausrad runter` | Workspace +1 (Monitor) |
| `SUPER + Mausrad hoch` | Workspace -1 (Monitor) |
| `SUPER + CONTROL + Mausrad hoch` | Workspace +1 (Monitor) |
| `SUPER + CONTROL + Mausrad runter` | Workspace -1 (Monitor) |
| `SUPER + SHIFT + S` | Fenster in Special Workspace (Scratchpad) verschieben |
| `SUPER + S` | Special Workspace (Scratchpad) umschalten |
